Animating CSS Filters

  • Animating greyscale on hover

          .grayscale {
        -webkit-filter: grayscale(1);
        filter: grayscale(1);
        transition: 2s;
        animation-timing-function:ease-in-out;
      }
      
      .grayscale:hover {
        -webkit-filter: grayscale(0);
        filter: grayscale(0);
      }

  • Animate on click 

          .animate-blur {
        -webkit-animation: blur-animated 2s linear;
      }
      
      @-webkit-keyframes blur-animated {
        0% {
          -webkit-filter: blur(0);
        }
        
        100% {
          -webkit-filter: blur(5px);
        }
      }
